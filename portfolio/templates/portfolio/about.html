The Human

Front End : Self-Taught Programmer
Framework : Problem-Solving & Global Sustainability
Platform : Strong Drive to Build Better [Everything]


After four years of living in China and experiencing the Airpocalypse of industrial smog in 2013 in Tianjin, I returned to the USA to pursue environmental stewardship. It quickly became obvious that environmental academics were not invoking any real change, so I shifted gears and pursued Geospatial Information Systems and software development. Scientists have identified the problems, I want to utilize data and technology to solve them.

I am currently a student at Portland Community College in Oregon. I have acquired the GIS Certificate and will be finished with the Associates of Science Degree promptly. Immediately after I will pursue an accelerated Bachelors in Computer Science from WGU, along with numerous associated tech certificates.

Currently I am part of the management team of a pandemic-surviving restaurant in Portland, and am eager to bring my systems-oriented operational solutions skills to the world of software. I am open for and seeking internships, full-time positions, and collaborative projects in solutions-oriented software, especially with organizations utilizing geospatial data.

Please view my résumé and the Maps and Code sections of this website for a more detailed sampling of my work.

The Website

Front End : Custom Code
Framework : Django
Platform : Heroku


This website is a custom design with all HTML, CSS, and Javascript written by yours truly. I use the python-based backend framework Django so that I can run locally developed python code on the web. This is all pushed through Git version control to the Linux-based Heroku platform.

Server-side debugging is performed in a Windows terminal accessing the bash terminal on Heroku. Both local and production apps are connected to a PostgreSQL database, that accepts data from a web-scraping python script.

All artwork on this site was also done by yours truly including the vectorized photo, the button backgrounds, the map designs etc. The "Code" button background on the homepage is actually the result of a python script and the python script itself that generates that very-psuedo-code imitation of binary smashed together in Adobe Illustrator. I am proficient with many Adobe products, but now use an amalgam of open-source and licensed software such as Affinity Designer and QGIS.
