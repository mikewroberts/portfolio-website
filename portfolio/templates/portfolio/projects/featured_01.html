Arduino Temperature Display

Simple Arduino project to warn user about a fish tank's water temperature if it goes out of range.


View code at : Github Repo

Project Details

Purpose:

In our household we have a pet axolotl.
They are extremely sensitive to environmental conditions.
I built this temperature display so that we can easily see if the water temp is in range, without having to remember the range itself.

I used an Arduino instead of buying a water sensor with a display pre-built, firstly because it's fun, and secondly because later on I would like to experiment with adding WiFi connectivity and pulling weather data with a server-side web scraper, and seeing how it affects the tank's temperature.

Hardware Setup:

Project consists of a water sensor with a stainless steel housing
that is safe for axolotls.
An Arduino Uno R3 from Elegoo is the microcontoller.
There's also an LCD with a potentiometer, and a currently unused WiFi module.
Full details, including where to buy these parts, can be found in the github repository.
