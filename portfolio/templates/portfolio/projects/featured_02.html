CO2 Emissions Webmap

Javascript interactive webmap that displays the CO2 emissions details of the countries with the highest emissions in 2016.

View code at : Github Repo
Project Details

Purpose:

This project is intended to provide an easy and interactive way to see which countries are emitting the most CO2.

Code:

This project mainly uses Javascript, CSS and HTML.
JQuery is utilized, as well as APIs from Stamen (for the basemap tile layer) and Leaflet (for the map interactivity).
The geographic data is held in geoJSON vector files, created with QGIS.

Site designed and programmed by Mike W Roberts.
