Genetic Odds Calculator

This is a calculator that produces a table of all possible genetic results from breeding a given pair of ball pythons and their traits.

Source code is proprietary to my client so it is not in a public github repo like my other featured projects.
Project Details

Client project.

A prototype was built first with Python in Jupyter Notebooks as a proof of concept. After proving the main algorithm works as well as the competitor's calculator, we moved into a web-based test page. That is the page linked to above.

The next step is to embed the Javascript in a Wordpress site.

Site designed and programmed by Mike W Roberts.
