Transformed a standalone script, written by Al Mowbray at the Metro Data Research Center, into an integrated ArcPro Python Tool to ship to other departments to use in various GIS analyses. Grid Map Procedural Generator

Program that can be used to generate a tile-based map for various end-uses (video games). “Rooms” generated with random shapes, sizes, and placements with an edge-detection moving-window algorithm running afterwards to delineate “Walls”. Built with C# and connected to Unity to use textfile to generate 3D blocks. In effect extrudes the 2D textfile into 3D game world.
